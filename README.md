# MeetSphere

A real-time video conferencing platform built using MERN, WebRTC and Socket.IO.

## Live Demo
https://meetsphere-video-conferencing-platf.vercel.app

## Features

- User Authentication (JWT)
- Create and Join Meetings
- Real-time Video Calling
- Real-time Chat
- Participant Counter
- Responsive UI

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Socket.IO
- WebRTC
- Tailwind CSS
